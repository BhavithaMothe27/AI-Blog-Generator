📝 AI Blog Generator

An AI-powered Blog Generator developed using **Python**, **Google Gemini**, and **Gradio**. The application creates high-quality, well-structured blog posts based on user inputs such as topic, audience, language, tone, terminology level, and desired word count.

---

🚀 Features

- 📝 Generate professional blogs on any topic
- 🌍 Supports multiple languages
- 🎭 Multiple writing tones
- 👥 Audience-specific content generation
- 📚 Adjustable terminology level
- 📏 Customizable word count
- 🤖 Powered by Google Gemini AI
- 🌐 Interactive Gradio web interface

---

🛠️ Technologies Used

- Python
- Google Gemini API
- Google Generative AI SDK
- Gradio

⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/BhavithaMothe27/AI-Blog-Generator.git

cd AI-Blog-Generator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

▶️ Run the Application

```bash
python app.py
```

---

 Configure Gemini API

Replace:

```python
genai.configure(api_key="YOUR_GEMINI_API_KEY")
```

with your own Google Gemini API key.

---

🖥️ Workflow

1. Enter the blog topic.
2. Specify the target audience.
3. Select the preferred language.
4. Choose the writing tone.
5. Select the terminology level.
6. Adjust the desired word count.
7. Click **Submit**.
8. Gemini generates a well-structured blog.
